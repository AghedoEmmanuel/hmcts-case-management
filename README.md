# HMCTS Case Management System

A full-stack task management application built for HMCTS, enabling case workers to create, view, update, and delete case-related tasks.

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | Next.js 14, TypeScript, Tailwind CSS, ShadCN UI |
| Backend | Node.js, Express, TypeScript |
| Database | PostgreSQL |
| Testing | Jest, React Testing Library, Supertest |
| DevOps | GitHub Actions (CI), dotenv |

## Project Structure
```
hmcts-case-management/
├── frontend/         # Next.js app
├── backend/          # Express API
├── .env              # Environment variables
└── README.md
```

## Features

- Create new case tasks
- View all tasks in a dashboard
- Update task status and details
- Delete tasks
- RESTful API with full CRUD operations
- Unit tested frontend components and backend routes
- PostgreSQL database with schema migrations
